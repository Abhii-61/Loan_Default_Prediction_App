# 💳 Loan Default Prediction App

A Machine Learning-based Loan Default Prediction App built using **Python**, **Scikit-learn**, and **Streamlit** that predicts whether a loan applicant is likely to default based on financial and personal information.

The project demonstrates an end-to-end Machine Learning workflow including data preprocessing, feature engineering, model training, evaluation, and deployment.

# ⚙️ How It Works

The application predicts whether a customer is likely to default on a loan based on user inputs.

## Workflow:
1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Encoding Categorical Variables
5. Feature Scaling
6. Model Training
7. Model Evaluation
8. Deployment using Streamlit

- Column Transformer
